Introduction

To use most of the Doot build tools, Node.js LTS version is required. Some of the plugins and framework in Doot does not yet support the latest Node.js version.

Doot is a fully featured premium chat app template in React Redux Saga with Firebase/Fake-backend with Facebook and Google Authentication and developer-friendly codes. We have not used jQuery in this template it's a pure ReactJs admin template with components based only. It's created with TypeScript.

Google and Facebook Authentication is not saved in Firebase/Fake-backend. It's just working with local storage.

Doot is a chat app template with beautifully crafted, clean & minimal designed with Dark & Light Modes.

It will help your team move faster and save development costs and valuable time. If you’re a developer and looking for a chat application UI that is fully responsive with Bootstrap and React, Redux, Saga without jQuery then you are at the right place to start your project using Doot - React Responsive Bootstrap 5 Chat App.

Doot contain lots of new design widgets with responsiveness on all screens. Also, there are 7 different types of color schemes added. We have written standard and developer-friendly code to increase performance.

How to use docs

The Doot is fully built with the Reactstrap. You can find Reactstrap component's docs in their official documentation.

Let's install your project first, please visit the installation page for a detailed guide on how to install and run Doot.

To get started, you can check out the well-structured folder structure of our template to understand how it works and its functionalities.
